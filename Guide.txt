🚀 TRuslan Portfolio Guide  

> A complete step-by-step guide to run and customize your Next-js portfolio. For beginners — no coding required.

✅ Step 1: Install Required Tools  
Before starting, make sure you have:  
- Node.js (LTS): https://nodejs.org/  
- Visual Studio Code: https://code.visualstudio.com/  

✅ Step 2: Set Up the Project (ZIP Version)  
1. Download and extract the ZIP file  
2. Rename the folder if you want (e.g., TRuslan-portfolio)  
3. Open the folder in Visual Studio Code  
4. In VS Code, open a terminal: Terminal → New Terminal  
5. Run this command to install all required packages:  

   npm install  

✅ Step 3: Customize Your Portfolio Content  
🛠 All main content is editable in one file:  
📂 /data/index.ts  

You can customize:  
- 🔗 Navigation (navLinks)  
- 💼 Projects (myProjects)  
- 🧠 Work Experience (workExperiences)  
- 💬 Client Testimonials (clientReviews)  


Example:  
export const navItems = [
  { name: "About", link: "#about" },
  { name: "Projects", link: "#projects" },

];

🧑‍🎨 Change Your Name & Headline  
Edit your intro and name in:  
📂 /components/Hero.tsx  

<p>Hi! I&apos; TRuslan, a Full-Stack Developer.</p>


📧 Update Contact Email in About Section  
To change the contact email shown and copied:  
📂 /components/ui/BentoGrid.tsx  

Replace this:  
 const text = "ruslontoraboyev0@gmail.com";  line.no - 55 replace this email to yours

navigator.clipboard.writeText('your email');

🚀 Step 5: Run the Project Locally  
👉 Run this command in terminal:  
   npm run dev  

👉 Open your browser and go to:  
   http://localhost:3000  

🧾 Full Command List  
npm install     # Install dependencies  
npm run dev     # Start the development server
